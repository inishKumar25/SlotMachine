# 🎮 WebGL Build – Slot Machine Game

This folder contains the WebGL build of the Slot Machine game.

---

## ▶️ How to Run Locally

⚠️ IMPORTANT:  
You **cannot** run this game by double-clicking `index.html`.  
WebGL builds require a local server.

---

### ✅ Method 1 — Python (Easiest)

If you have Python installed:

#### Step 1: Open terminal in this folder
Navigate to:

```
cd Build/WebGL

```
Enter this in your terminal after locating to the Directory

python -m http.server 8000

Then go to this link :

http://localhost:8000
